# How To Install

```
curl -fsSL https://raw.github.com/AdrienGiboire/dotfiles/master/scripts/install > /tmp/install.sh && bash /tmp/install.sh && rm /tmp/install.sh
curl -fsSL https://raw.github.com/AdrienGiboire/dotfiles/master/scripts/install > /tmp/install.sh && bash /tmp/install.sh --node && rm /tmp/install.sh
curl -fsSL https://raw.github.com/AdrienGiboire/dotfiles/master/scripts/install > /tmp/install.sh && bash /tmp/install.sh --ruby && rm /tmp/install.sh
curl -fsSL https://raw.github.com/AdrienGiboire/dotfiles/master/scripts/install > /tmp/install.sh && bash /tmp/install.sh --node --ruby && rm /tmp/install.sh
```
